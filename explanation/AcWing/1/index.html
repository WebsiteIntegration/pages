AcWing 1.A+B    

2020年8月13日    A-LittleLuck

这道题真的好简单哦qwq
只要会读入+读出+变量定义就可以了
所以我们就直接来看代码吧: 

#include <iostream>
using namespace std;
int main()
{
    int a,b;   //定义a和b
    cin>>a>>b; //输入a和b
    cout<<a+b;   //输出a与b的和
    return 0;  //退出程序
}
